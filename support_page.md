## 🔧 자주 발생하는 문제 해결

### 1. 백업/복원 관련

#### Q: 백업 버튼을 눌렀는데 작동하지 않아요
**해결 방법:**
1. iOS 설정 → iScript → 권한 확인
2. 앱을 완전히 종료 후 재실행
3. 여전히 문제가 있다면 기기 재시작

#### Q: 복원이 무한 로딩되어요
**해결 방법:**
- 최신 버전(1.0.6 이상)으로 업데이트하세요
- FilePicker 플러그인이 개선되어 즉시 작동합니다

#### Q: 백업 파일을 찾을 수 없어요
**해결 방법:**
1. **파일 앱** 열기
2. iCloud Drive 또는 최근 항목 확인
3. 검색창에 "iScript_backup" 입력

---

### 2. AI 기능 관련

#### Q: AI 분석이 작동하지 않아요
**확인 사항:**
1. API 키가 정확히 입력되었나요?
   - 설정 → API Key 확인
2. 인터넷 연결이 되어 있나요?
3. API 키가 유효한가요?
   - [Google AI Studio](https://aistudio.google.com/app/apikey)에서 재확인

#### Q: TTS(발음) 기능이 작동하지 않아요
**해결 방법:**
1. 기기 음소거 해제
2. 볼륨 확인
3. 인터넷 연결 확인
4. 앱 재시작

---

### 3. 광고 관련

#### Q: 광고가 너무 자주 나와요
**현재 광고 표시 시점:**
- 배너 광고: 앱 시작 시 하단 고정
- 전면 광고: 대본 시작 시, Analytics 이동 시

> 광고는 무료 앱 운영을 위한 것이며, 향후 광고 제거 옵션을 추가할 예정입니다.

#### Q: 광고를 제거할 수 있나요?
**답변:**
- 현재는 무료 버전만 제공됩니다
- 향후 업데이트에서 광고 제거 옵션을 제공할 예정입니다

---

### 4. 대본 업로드 관련

#### Q: 엑셀 파일 업로드가 안 돼요
**확인 사항:**
1. 파일 형식이 .xlsx 또는 .xls인가요?
2. 1열: 원문, 2열: 번역문 형식인가요?
3. 저작권 동의 체크박스를 선택하셨나요?

#### Q: 업로드 후 문장이 이상하게 나와요
**해결 방법:**
1. 엑셀 파일을 다시 확인
2. 빈 셀이 없는지 확인
3. 셀 병합이 없는지 확인

---

### 5. 학습 데이터 관련

#### Q: 학습 기록이 사라졌어요
**원인:**
- 앱 삭제 시 로컬 데이터가 함께 삭제됩니다
- 백업 파일이 없으면 복구가 불가능합니다

**예방 방법:**
- 일주일에 한 번씩 백업 파일 생성
- iCloud Drive에 백업 파일 저장

#### Q: SRS 복습이 제대로 작동하지 않아요
**확인 사항:**
1. 기기 날짜/시간 설정이 자동인가요?
2. 문장을 "Learned"로 표시하셨나요?
3. Analytics에서 복습 일정 확인
