# **개인정보처리방침 (Privacy Policy)**
**iScript Lab**은 사용자의 개인정보를 소중히 다루며, 관련 법규를 준수합니다.



### 1. 수집하는 개인정보 항목
**iScript** 앱은 사용자의 어떠한 개인정보(이름, 이메일, 연락처 등)도 자체 서버로 전송하거나 수집하지 않습니다.
단, 광고 서비스(Google AdMob) 제공을 위해 다음과 같은 기기 식별 정보가 수집될 수 있습니다.
*   광고 식별자 (IDFA)

### 2. 데이터 저장 방식 (Local-First)
모든 학습 데이터와 대본 파일은 사용자의 기기 내부 저장소(IndexedDB)에만 저장됩니다. 앱을 삭제하면 데이터도 함께 삭제됩니다.

### 3. 외부 API 사용
AI 분석 및 음성 재생(TTS) 기능을 위해 텍스트 데이터가 **Google Gemini API**로 전송될 수 있습니다. 이 과정에서 사용자의 API Key가 사용되며, 개발자는 해당 데이터를 열람하거나 저장할 수 없습니다.

### 4. 개인정보의 수집 및 이용 목적
*   앱 내 광고 제공 (Google AdMob) 및 성과 분석
*   AI 기반 언어 학습 기능 제공 (Gemini API)

### 5. 개인정보의 제3자 제공
본 앱은 원칙적으로 사용자의 개인정보를 외부에 제공하지 않습니다. 단, 광고 서비스 제공을 위해 아래 업체에 식별 정보가 제공될 수 있습니다.
*   **제공받는 자**: Google AdMob
*   **제공 목적**: 맞춤형 광고 제공

### 6. 문의사항
문의사항이 있으시면 고객 지원 페이지를 통해 연락주시기 바랍니다.
