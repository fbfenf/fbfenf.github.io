# 개인정보처리방침 (Privacy Policy)


iScript Lab은 사용자의 개인정보를 소중히 다루며, 관련 법규를 준수합니다.

---

## 1. 수집하는 개인정보 항목

iScript 앱은 사용자의 어떠한 개인정보(이름, 이메일, 연락처 등)도 자체 서버로 전송하거나 수집하지 않습니다. 

단, 광고 서비스(Google AdMob) 제공을 위해 다음과 같은 기기 식별 정보가 수집될 수 있습니다:
- **광고 식별자 (IDFA)**: 맞춤형 광고 제공을 위해 사용됩니다.

---

## 2. 데이터 저장 방식 (Local-First)

### 2.1 로컬 저장
모든 학습 데이터와 대본 파일은 사용자의 **기기 내부 저장소(IndexedDB)**에만 저장됩니다. 

### 2.2 파일 백업 기능
- 사용자는 앱 설정에서 **'백업' 버튼**을 통해 학습 데이터를 **JSON 파일**로 저장할 수 있습니다.
- **Share 기능**을 통해 iCloud Drive, 다운로드 폴더 등 원하는 위치에 백업 파일을 저장할 수 있습니다.
- **파일 복원**: FilePicker를 통해 이전에 저장한 백업 파일을 선택하여 데이터를 복원할 수 있습니다.

### 2.3 데이터 삭제
앱을 삭제하면 기기에 저장된 모든 학습 데이터가 함께 삭제됩니다. 백업 파일을 생성하지 않은 경우 데이터 복구가 불가능합니다.

---

## 3. 외부 API 사용

### 3.1 Google Gemini API
AI 분석 및 음성 재생(TTS) 기능을 위해 텍스트 데이터가 **Google Gemini API**로 전송될 수 있습니다. 

- 사용자의 **개인 API Key(BYOK)**가 사용됩니다.
- 개발자는 해당 데이터를 열람하거나 저장할 수 없습니다.
- 전송되는 데이터: 학습 중인 문장 텍스트

### 3.2 Google AdMob
- **광고 표시**: 배너 광고(하단 고정)와 전면 광고(대본 시작 시, Analytics 화면 이동 시)가 표시됩니다.
- **광고 식별자(IDFA)** 사용: 맞춤형 광고 제공을 위해 사용됩니다.
- **사용자 제어**: 기기 설정(설정 → 개인정보 보호 → 추적)에서 맞춤형 광고를 제한할 수 있습니다.
