# **Brand Identity**

**iScript: intelligent Script for U&i**는 단순한 어학 도구를 넘어, 기술과 사람의 유기적인 성장을 지향합니다.

- **U (User)**: 학습의 주체인 당신
- **& (Connection)**: 서로 소통하며 성장하는 연결
- **i (Intelligence/Individual)**: 최신 AI 기술과 개별 기록의 조화

### **1. AI 학습 기능 및 API 설정** 🤖

iScript의 강력한 분석 기능을 사용하려면 개인용 API 키가 필요합니다.

- **Gemini API 키 발급 (무료)**: [Google AI Studio 바로가기](https://aistudio.google.com/app/apikey)
- **정밀 분석 (AI Explain)**: 발급받은 키를 앱 설정에 입력하면 문법, 핵심 단어, 문화적 배경까지 상세히 설명해 드립니다.
- **오프라인 지원**: 고성능 AI TTS를 통해 원어민 발음을 듣고, 로컬 캐싱 기능을 통해 지하철에서도 데이터 걱정 없이 공부하세요.

### **2. 자료 추가 및 테마 설정** 📂

iScript는 사용자가 소유한 엑셀 파일(.xlsx)을 학습 데이터(단어장, 대본)로 사용하는 **'지능형 플레이어'**입니다.

- **대본 얻는 법**: 직접 제작한 단어장이나 대본, 공유받은 엑셀 파일을 앱으로 불러오세요. 상세 제작은 위 **'유튜브 가이드'**를 참고해 주세요.
- **엑셀 파일 구조**

| **구분** | A열 (1열) | B열 (2열) |
| --- | --- | --- |
| 내용 | 학습할 외국어 단어, 문장 | 모국어 해석 (뜻) |
| 예 | Can I get a coffee? | 커피 한잔 주시겠어요? |

### **3. 주요 학습 기능** 📚

- **🔊 AI 원어민 발음 (TTS)**: 스피커 아이콘을 클릭하여 원어민 음성을 듣습니다. 아이콘 위 **숫자 배지**는 반복 청취 횟수입니다.
- **✨ AI Explain**: 문장 분석을 통해 주요 단어의 뜻과 발음을 제공합니다. 별표(⭐)를 눌러 단어를 북마크하세요.
- **🙈 블러 집중 (Blur Focus)**: 번역문은 기본적으로 블러 처리되어 있습니다. 스스로 해석해 본 뒤 영역을 클릭하여 정답을 확인하세요. (Coming Soon)
- **🎨 프리미엄 테마**: 눈이 편안한 `Sepia` 테마를 통해 학습 몰입도를 높일 수 있습니다.

### **4. 스마트 복습 (SRS)** 🧠

- **망각 곡선 알고리즘**: 당신의 복습 타이밍을 완벽하게 계산하여, 최적의 시점에 학습 문장을 다시 노출해 드립니다.

| **상태** | **설명** | **작동 방식** |
| --- | --- | --- |
| **NEW** | 신규 문장 | 처음 업로드되어 학습 전인 상태 |
| **LEARNED** | 학습 완료 | '완료' 버튼을 누르면 복습 주기에 진입 |
| **REVIEW** | 복습 필요 | 알고리즘에 의해 자동으로 화면에 다시 노출 (1일→3일→7일...) |
