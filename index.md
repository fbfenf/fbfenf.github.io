## 🚀 빠른 시작

### iScript란?

iScript는 **사용자가 보유한 대본 파일(엑셀)**을 활용하여 외국어를 효과적으로 학습할 수 있도록 돕는 **AI 기반 iOS 학습 도구**입니다.

### 어떻게 활용하나요?

1. **🎬 문장 학습**: 영화/미드 대본으로 생생한 표현 익히기
2. **📝 나만의 단어장**: 엑셀(1열 단어, 2열 뜻)만 있으면 끝!
3. **🧠 스마트 암기**: SRS(간격 반복 시스템)로 잊을 때쯤 다시 복습

## ✨ 주요 기능

### 1. 강력한 AI 학습 도구 🤖

- **🔊 원어민 AI 발음 (TTS)**: 스피커 아이콘을 눌러 자연스러운 발음을 들어보세요.
- **✨ AI 문장 분석 (Explain)**: 문법, 어휘, 문화적 배경까지 한 번에 설명해줍니다.
- **⭐ 단어 북마크**: 모르는 단어는 별표를 눌러 저장하고 따로 복습하세요.

> **Tip**: AI 기능을 100% 활용하려면 **무료 API 키**가 필요합니다.  
> 👉 [**API 키 발급 가이드 보기**](/api)

### 2. 편리한 대본 관리 📂

- **엑셀 파일(.xlsx) 지원**: A열(원문), B열(번역문) 형식의 파일만 있으면 됩니다.
- **쉬운 편집**: 이름 변경, 삭제, 문장 추가 등 자유롭게 수정 가능
- **샘플 제공**: 대본이 없어도 걱정 마세요! [자료실](/resources)에서 무료 샘플을 받아보세요.

### 3. 집중력을 높이는 학습 모드 🎯

- **블러링(Blur) 학습**: 번역문을 가리고 스스로 해석해보세요. 탭하면 정답이 보입니다.
- **학습 상태 관리**:
  - <span style="color: blue">🔵 **New**</span> : 처음 본 문장
  - <span style="color: green">🟢 **Learned**</span> : 확실히 아는 문장
  - <span style="color: orange">🟠 **Review**</span> : 복습이 필요한 문장 (자동 알림)

## 🛠️ 고급 기능 가이드 (더보기)

<details>
<summary><strong>📂 데이터 백업 및 복원 방법 (클릭하여 펼치기)</strong></summary>

<br>
iScript는 <strong>로컬 저장소</strong>를 사용하므로, 앱 삭제 시 데이터가 함께 삭제됩니다. <strong>정기적인 백업</strong>을 권장합니다!

#### 1. 백업하기 (데이터 저장)
1. **설정** 탭 이동
2. "파일 백업" 섹션에서 **[백업]** 버튼 탭
3. 저장 위치 선택 (추천: **iCloud Drive**, 파일 앱, 메일 등)
4. `.json` 파일이 안전하게 저장됩니다.

#### 2. 복원하기 (데이터 불러오기)
1. **설정** 탭 이동
2. "파일 백업" 섹션에서 **[복원]** 버튼 탭
3. 저장해둔 `.json` 파일 선택
4. 앱이 재시작되며 데이터가 복구됩니다.
</details>

<br>

<details>
<summary><strong>📢 광고 및 무료 정책 안내 (클릭하여 펼치기)</strong></summary>

<br>
iScript는 <strong>완전 무료</strong>로 제공되며, 서버 유지보수를 위해 최소한의 광고가 포함되어 있습니다.

- **배너 광고**: 화면 하단에 작게 표시됩니다.
- **전면 광고**: 대본 학습을 마치고 홈으로 돌아갈 때만 가끔 표시됩니다. (학습 중에는 방해하지 않습니다!)
- **광고 제어**: iOS 설정 > 개인정보 보호 > 추적 설정에서 맞춤형 광고를 제어할 수 있습니다.
</details>

<br>

<details>
<summary><strong>🎨 테마 및 언어 설정 (클릭하여 펼치기)</strong></summary>

<br>
#### 테마 변경
- **Light**: 깔끔하고 밝은 화면
- **Dark**: 눈이 편안한 어두운 화면 (배터리 절약!)
- **Sepia**: 종이책 같은 따뜻한 느낌

#### 언어 설정
앱 내 설정에서 **한국어(KO)** 또는 **영어(EN)**를 선택할 수 있습니다.
</details>

## 🔒 개인정보 및 보안

<details>
<summary><strong>내 데이터는 어디에 저장되나요? (안전해요!)</strong></summary>

<br>
iScript는 <strong>"Local-First"</strong> 원칙을 따릅니다.

1. **내 기기에만 저장**: 모든 학습 기록과 대본은 사용자 기기(iPhone/iPad) 내부에만 저장됩니다.
2. **서버 전송 없음**: iScript 개발팀은 여러분의 대본 내용을 볼 수 없습니다.
3. **AI 분석**: 문장 분석 시에만 사용자의 API Key를 통해 AI 서비스로 텍스트가 전송되며, 저장되지 않고 분석 후 즉시 폐기됩니다.

자세한 내용은 [개인정보처리방침](/privacy/)을 확인하세요.
</details>
