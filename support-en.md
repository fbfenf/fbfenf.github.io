---

### 4. Script Upload

#### Q: Excel file won't upload

**Check:**

1. Is file format .xlsx or .xls?
2. Is format correct (Column 1: Original, Column 2: Translation)?
3. Did you check the copyright agreement checkbox?

#### Q: Sentences appear incorrectly after upload

**Solution:**

1. Recheck Excel file
2. Check for empty cells
3. Check for merged cells

---

### 5. Learning Data

#### Q: Learning records disappeared

**Cause:**
- Local data is deleted when app is deleted
- Recovery impossible without backup files

**Prevention:**
- Create backup files once a week
- Save backup files to iCloud Drive

#### Q: SRS review doesn't work properly

**Check:**

1. Is device date/time set to automatic?
2. Did you mark sentences as "Learned"?
3. Check review schedule in Analytics
