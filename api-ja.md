# 🔑 AI API Key 取得ガイド

iScriptの強力なAI機能（発音リスニング、文章分析など）を使用するには、無料の**AI API Key**が必要です。
Googleアカウントがあれば、誰でも**無料**で簡単かつ迅速に取得できます。

---

## 🚀 3分で取得する方法（やってみよう！）

### ステップ 1: APIキー発行ページへ
下のボタンをクリックして、GoogleのAPIキー管理ページに移動します。

👉 [**Google AI Studioへ移動**](https://aistudio.google.com/app/apikey)

> **注**: Googleアカウントでのログインが必要な場合があります。

### ステップ 2: APIキーを作成する
1. 画面にある青い **Create API key**（APIキーを作成）ボタンをクリックします。
2. ポップアップメニューから **Create API key in new project**（新しいプロジェクトでAPIキーを作成）を選択します。
   *(既存のプロジェクトがある場合は、それを選んでも構いません)*

### ステップ 3: キーをコピーする
1. APIキーが作成されます。
2. 作成されたキーの隣にある **Copy**（コピー）ボタン（書類アイコン）を押してコピーします。
   *(キーは `AIza` で始まる長い文字列です)*

### ステップ 4: iScriptアプリに入力
1. **iScriptアプリ**を開きます。
2. **Settings**（設定）メニューに移動します。
3. **API Key**（APIキー）項目をタップし、コピーしたキーを貼り付け（Paste）ます。
4. **Save**（保存）して完了です！ 🎉

---

## 💡 よくある質問 (FAQ)

### Q: これは本当に無料ですか？
**はい、無料です。** 個人ユーザーは、Google AI Studioの無料枠（Free Tier）を通じて十分な量を無料で利用できます。

### Q: クレジットカードの登録は必要ですか？
いいえ、無料枠を利用する場合、**クレジットカードの登録は不要です。**

### Q: APIキーが機能しません。
- コピー時に空白が含まれていないか確認してください。
- キーを再発行して、もう一度入力してみてください。
